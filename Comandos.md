# Comandos

```bash
date # Mostrar la fecha actual
echo "Hello World!" # Imprimir mensaje en pantalla
man date # Manual de x comando
history # Historial de los comandos usados
pwd # Print Working Directory. Imprime el directorio actual
cd - # Cambiar al ultimo diectorio visitado

# LS
ls -l # Listar propiedades de los archivos/directorios. Si empieza con una 'd' es un directorio.
ls -lh # Muestra la misma información que ls -l pero con las unidades de tamaño en KB, MB
ls -t # Ordena los archivos por fecha de modificación
ls -x # Ordena elementos primero por nombre y después por extensión
ls -X # Ordena los elementos primero por extensión y luego por nombre:
ls -R # Muestra el contenido de todos los subdirectorios de forma recursiva
ls - S # Ordena los resultados por tamaño de archivo

rmdir folder_mame # Eliminar un directorio. No puede borrar directorios con archivos dentro.
more file.txt # Divide una salida larga en varias páginas
wc # (Word Count) Contar, palabras, caracteres, etc
wc -l file.txt # Contar cuantas líneas tiene el archivo.
```
